Quick Integration

1. Get Recommendations

curl -X POST https://api.vfied.com/v1/recommend \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{
"location": {
"city": "London",
"country": "UK",
"country_code": "GB"
},
"mood_text": "tired but celebrating",
"dietary": ["vegetarian"]
}'

2. Upload Your Menu

curl -X POST https://api.vfied.com/v1/menus \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{
"menu": [{
"name": "Ugali Mayai",
"country_code": "KE",
"tags": ["vegetarian", "protein-rich"],
"price": "8.99"
}]
}'
